# SCP Smart Contracts — Documentation Repository

This repository is dedicated exclusively to documentation for the SCP Smart Contracts project. It contains technical references, architecture guides, API documentation, and user guides.

## Repository Structure

```
docs/
├── architecture/   # System architecture and design documents
├── contracts/      # Smart contract specifications and references
├── guides/         # User and developer guides
└── api/            # API references and integration guides
```

## Getting Started

Browse the [`docs/`](./docs/) directory to find the documentation you need:

- **Architecture** — High-level design decisions and system diagrams
- **Contracts** — Individual smart contract specifications, ABI references, and deployment guides
- **Guides** — Step-by-step guides for users and developers
- **API** — REST / RPC API references and integration examples
